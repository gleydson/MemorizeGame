## :rocket: Content

The following contents covered in this course that were applied in this project:

- Intro to SwiftUI
- MVVM and the Swift Type System
- Reactive UI Protocols Layout
- Enums and Optionals
- ViewBuilder Shape ViewModifier
- Animation
